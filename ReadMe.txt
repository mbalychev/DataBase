Пример проекта. Сайт для организации, в наличии которой персональные компьютеры и планшеты. В штате рганизации менеджеры и обслуживающий персонал. 
Обслуживающий персонал производит ремонт компьютеров и планшетов, отмечая в ремонтном листе изменения. 
Проект Asp.NET core MVC. Entity Framework Core (EF Core). Asp.Net core identity. LINQ. Методология ООП.
